⚡ Core Features
Everything you need to
dominate dealer intel
From raw scraping to enriched, scored profiles — the full pipeline in one platform.
🔍
Multi-Channel Scraping
Simultaneously pulls dealer data from Google Maps, Mudah.my, Carlist.my and more — covering 7 live channels with zero mock data.
📧
Contact Enrichment
Automatically discovers phone numbers, emails, websites and WhatsApp links for every dealership profile found.
🤖
AI Scoring Engine
Every dealer is ranked with an AI-driven score based on data completeness, channel presence and business signals.
📣
Ad Campaign Manager
Dealers can upload media, set budgets in MYR, and define campaign durations — all tracked in a dedicated dashboard.
💼
LinkedIn Prospecting
Cross-reference dealership names with LinkedIn to find key decision-makers and sales contacts automatically.
🗄️
Supabase Cloud Storage
All enriched dealer data, campaigns and user accounts are stored in real-time in a managed Supabase Postgres database.
